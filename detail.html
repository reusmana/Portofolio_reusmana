Project Informasi

Category : Pencatatan Keuangan Desa
Client : Desa Gunung Malang Cikidang Sukabumi
Project Date : 22 Mei 2020
Project Source Code : Github

Description

Aplikasi dibuat untuk melakukan Pencatatan pengeluaran dan pemasukan di desa, dengan 4 kategori Keuangan, yaitu DD (Dana Desa), ADD (Anggaran Dana Desa), BanKeu (Bantuan Keuangan), dan BanProv (Bantuan Provinsi).
Terdapat Fitur Grafik Perbulan total pengeluaran dan total pemasukan, Saldo, Rekap, Cetak ke pdf or csv, dll
